# How to use
* Install npm packages
```bash
npm install
````

* copy "Resouces.resx" file in current folder
* Run script
```bash
node app.js
```
* The tool will generate translation the following target language files in current folder
```text
Resouces_de.resx
Resouces_fr.resx
Resouces_ja.resx
Resouces_ko.resx
Resouces_zh-cn.resx
Resouces_zh-tw.resx
```
